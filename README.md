# Responsive Login & Registration System

A responsive Login and Registration System built using HTML, CSS, Bootstrap, JavaScript, PHP, and AJAX.

## 📌 Project Overview

This project was developed for Interactive UI & Frontend Development. It demonstrates responsive web design, form validation, AJAX integration, and basic PHP backend functionality.

## 🚀 Features

- Responsive and Mobile-Friendly Design
- Login Form
- Registration Form
- Form Validation using JavaScript
- Password Match Validation
- Show/Hide Password Toggle
- AJAX Email Availability Check
- PHP Form Processing
- Success and Error Alerts
- Reusable Navbar and Footer Components
- Bootstrap 5 UI Components

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- AJAX
- PHP

## ⚙️ How to Run the Project

### Prerequisites

- XAMPP or any local PHP server
- Web Browser

### Steps

1. Clone the repository

```bash
git clone https://github.com/AmruthavarshiniAvvari/Secure_Login_System.git
```

2. Move the project folder into the XAMPP htdocs directory

```plaintext
C:\xampp\htdocs\
```

3. Start Apache from XAMPP Control Panel.

4. Open your browser and visit:

```plaintext
http://localhost/TASK2/
```
